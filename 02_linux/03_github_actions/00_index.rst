========================
Github Actions for CI/CD
========================

Github Actions are integrated tools in GitHub for Continuous Integration (CI)
and Continuous Delivery (CD), i.e. it allows to automatically build, test and deploy
software projects. For more details refer to the `official documentation <https://docs.github.com/en/actions>`_.
